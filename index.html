Tugas 1 - Dasar Algoritma Pemrograman

🏧 Mengambil Uang di ATM

  1. Masukkan kartu ATM.
  2. Pilih bahasa.
  3. Masukkan PIN.
  4. Pilih menu "Tarik Tunai".
  5. Masukkan jumlah uang.
  6. Konfirmasi transaksi.
  7. Ambil uang.
  8. Ambil kartu dan struk.
  9. Selesai.

☕ Membuat Kopi Manis dan Pahit yang Pas

  1. Siapkan gelas, kopi, gula, dan air panas.
  2. Masukkan 1–2 sendok kopi.
  3. Tambahkan gula secukupnya.
  4. Tuang air panas.
  5. Aduk hingga rata.
  6. Cicipi kopi.
  7. Jika terlalu pahit, tambahkan gula.
  8. Jika terlalu manis, tambahkan kopi atau air.
  9. Aduk kembali dan sajikan.
  10. Selesai.

🧮 Menghitung Nilai Y = 3x + 6

  1. Masukkan nilai x.
  2. Hitung 3 × x.
  3. Tambahkan hasil dengan 6.
  4. Dapatkan nilai y.
  5. Tampilkan hasil.

Contoh: Jika x = 2 maka y = (3 × 2) + 6 = 12

📚 Pengertian Algoritma dan Program

Algoritma adalah langkah-langkah logis dan sistematis yang digunakan untuk menyelesaikan suatu masalah.

Program adalah kumpulan instruksi (kode) yang ditulis menggunakan bahasa pemrograman untuk menjalankan algoritma pada komputer.

🔄 Struktur Dasar Algoritma

A. Sekuensial (Urutan)

Langkah-langkah dikerjakan secara berurutan dari awal sampai akhir.

Contoh: Memasak mie instan.

B. Pemilihan (Selection)

Menggunakan kondisi IF-ELSE untuk menentukan keputusan.

  • Jika hujan → pakai payung.
  • Jika tidak hujan → tidak perlu payung.

C. Perulangan (Looping)

Mengulang suatu proses sampai kondisi tertentu terpenuhi.

  • Mengaduk kopi sampai gula larut.
  • Belajar sampai paham.

💻 Fungsi Bahasa Pemrograman

  • Membuat program atau aplikasi.
  • Mengatur kerja komputer.
  • Mengolah data.
  • Membantu otomatisasi pekerjaan.
  • Menjadi penghubung antara manusia dan komputer.

Tugas 2 - Tipe Data dan Operator Java

📚 1. Perbedaan Tipe Data Primitif dan Non-Primitif

Tipe Data Primitif

Tipe data primitif merupakan tipe data dasar yang hanya menyimpan satu nilai dalam satu variabel.

  • Integer (int)
  • Float
  • Boolean
  • Char

Tipe Data Non-Primitif

Tipe data non-primitif merupakan tipe data yang dapat menyimpan lebih dari satu nilai dan biasanya dibuat atau didefinisikan oleh programmer.

  • String
  • Array
  • Interface
  • Enum
  • Class

Tugas 3 - Struktur Algoritma Sekuensial

📌 1. Pengertian Instruksi pada Algoritma

Instruksi pada algoritma adalah perintah atau langkah-langkah yang diberikan secara berurutan untuk menyelesaikan suatu masalah atau menjalankan suatu proses.

📖 2. Pengertian Struktur Algoritma Sekuensial

Struktur algoritma sekuensial adalah struktur algoritma yang dijalankan secara berurutan mulai dari langkah pertama hingga langkah terakhir tanpa adanya percabangan maupun perulangan.

🔄 3. Karakteristik Struktur Sekuensial

  • Instruksi dijalankan secara berurutan.
  • Proses dimulai dari atas ke bawah.
  • Tidak terdapat percabangan (selection).
  • Tidak terdapat perulangan (looping).
  • Setiap langkah hanya dilakukan satu kali.
  • Hasil proses bergantung pada urutan langkah yang diberikan.

💻 4. Contoh Struktur Sekuensial

Contoh sederhana penggunaan struktur sekuensial adalah menghitung luas persegi panjang. 

panjang = 10
lebar = 5

luas = panjang * lebar

print("Luas persegi panjang =", luas)

Hasil Output

Luas persegi panjang = 50

Tugas 4 - Menghitung Total Belanja

🛒 Studi Kasus

Program ini digunakan untuk menghitung total belanja berdasarkan jumlah barang yang dibeli dan harga masing-masing barang. Setiap harga barang akan dijumlahkan sehingga diperoleh total keseluruhan belanja.

📊 Flowchart

📝 Pseudocode

Algoritma Menghitung Total Belanja

Deklarasi:
n, i : integer
harga, total : integer

Deskripsi:

Input jumlah barang

total ← 0

Untuk i ← 1 sampai n lakukan

    Input harga barang

    total ← total + harga

Akhir Untuk

Tampilkan total belanja

💻 Program Python

# Program Menghitung Total Belanja

n = int(input("Masukkan jumlah barang: "))

total = 0

for i in range(n):

    harga = int(
        input("Masukkan harga barang: ")
    )

    total = total + harga

print("Total belanja =", total)

Tugas 5 - Array dalam Pemrograman

📚 1. Pengertian Array

Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data dengan tipe yang sama dalam satu variabel dan diakses menggunakan indeks.

🔢 2. Array Satu Dimensi (1D)

Array satu dimensi adalah array yang hanya memiliki satu baris atau satu indeks untuk mengakses elemennya.

Contohnya seperti daftar nilai siswa dalam satu deretan. 

nilai = [80, 85, 90, 75, 88]

📝 3. Cara Mendeklarasikan Array

Array dideklarasikan dengan menentukan nama array dan mengisi elemen-elemen yang akan disimpan. 

angka = [10, 20, 30, 40, 50]

💻 4. Tipe Data yang Digunakan pada Array

Array dapat menyimpan berbagai tipe data, antara lain:

  • Integer (Bilangan Bulat)
  • Float (Bilangan Desimal)
  • String (Teks)
  • Boolean (True / False)
  • Character (Karakter)

Contoh:

umur = [18, 19, 20]      # Integer

nama = ["Andi", "Budi"]  # String

📦 5. Macam-Macam Array

  • Array Satu Dimensi (1D)
  • Array Dua Dimensi (2D)
  • Array Tiga Dimensi (3D)
  • Array Multidimensi

Contoh Array Dua Dimensi

matriks = [

    [1, 2, 3],

    [4, 5, 6]

]
